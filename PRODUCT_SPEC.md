# Shared Disposable Camera MVP

Platform:
- iOS only

Goal:
Let a small group create an event, take limited in-app photos, hide them until unlock time, then reveal a shared album.

Core Features:
- Create event
- Join event via code/link
- Limited shots per member
- In-app camera only
- No viewing photos before unlock
- No deleting photos
- Album reveal after unlock time
- Local-first photo saving
- Sync to backend later

Non-goals:
- Android
- PWA
- Web app
- Film filters
- Social feed
- Likes/comments
- Payments
- AI curation
- Photobook export
